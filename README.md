# Advent of Code 2025

Based on .NET 10.

This project contains solutions for the Advent of Code 2025 challenges.

## How to Run

To run the project and select a specific day and input file:

1.  **Run the application:** Execute the following command in the terminal:
    ```bash
    dotnet run --project AdventOfCode
    ```

2.  **Follow the prompts:** The application will display a list of available days (folders under `TestInputes`).
    *   Enter the number corresponding to the day you wish to process.
    *   After selecting a day, a list of available input files for that day will be shown.
    *   Enter the number corresponding to the file you wish to process.

    The program will then execute the processing logic for the selected day and file.
